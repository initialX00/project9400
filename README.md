
## ✨ 프로젝트 소개 [Notion](https://www.notion.so/Project9400-1cb02444254a80b2a9a0c9a6dc9c6e23)

### 목적

-   불편함이 크다고 알려진 맥도날드 키오스크를 개선하기 </br></br>


### 팀 프로젝트

-   팀원 4명

-   2025.03.10 ~ 2025.04.11  </br></br>

### 주요 기능

-   관리자 회원가입 및 로그인, oAuth로 회원가입 및 로그인
-   관리자 모드 메뉴

    > 메뉴 조회, 메뉴 노출 변경
    
    > 메뉴 추가, 수정, 삭제
    
    > 전체 매출 및 제품별 매출 조회

    > 결제 내역 조회 및 결제 취소

    > 관리자 정보 수정 (점포명 변경, 비밀번호 변경, 이메일 변경)

-   사용자 모드 메뉴

    > 포장/매장 여부 선택, 언어 선택 

    > 메뉴 선택 ( 단품, 세트 / 미디엄, 라지 / 세트메뉴의 구성품 )
    
    > 장바구니 수량 추가, 삭제, 변경

    > 결제 방법(카카오페이)

    > 포인트 적립/사용

 </br></br> </br></br>


## 💼 역할 분담

<h3> 팀장: 권민창 </h3>

### 🔧 Back-End
- 제품 추가
- 제품 정보 변경
- 제품 삭제
- 원산지,영양정보 변경

### 🖥 Front-End
- 관리자 제품 관리 페이지
- 관리자 원산지/영양정보 관리 페이지


-   프로젝트 최종 발표
</br></br>

<h3> 팀원: 백진우 </h3>

### 🔧 Back-End

- 메뉴 조회
- 메뉴 노출 변경
- OAuth2 회원가입, 로그인


### 🖥 Front-End

- 관리자 로그인, 회원가입 페이지
- 관리자 메뉴 관리 페이지
- 관리자 매출 조회 페이지 (portone)
- 고객 메뉴 상세정보 조회 

</br></br>

<h3> 팀원: 최명준 </h3>

### 🔧 Back-End

-   OAuth2 회원가입
-   security
-   포인트 관리
-   관리자 정보 변경
-   사용자 등록

### 🖥 Front-End

-   관리자 회원가입, 로그인 페이지
-   관리자 정보 변경 페이지
-   관리자 매출 조회 페이지
-   고객 포인트 페이지
</br></br>

<h3> 팀원: 최석현 </h3>

### 🔧 Back-End
-   메뉴 불러오기
-   주문 정보를 저장

### 🖥 Front-End
-   고객 홈 페이지(언어, 장소)
-   고객 메뉴 페이지(카테고리별 메뉴)
-   고객 주문을 조회하는 장바구니 페이지
-   고객 결제 페이지
-   고객 주문번호 출력 페이지
-   관리자 카테고리 관리 페이지
</br></br>

<h3> 공동 작업 </h3> 

-   디자인 도안 작업
-   API 명세서 작성
-   DB 설계
-   ERD 작성
</br></br></br></br>


## 🤝 협업 방식

1. 팀원 간의 명확한 아이디어 공유를 위해 figma와 front로 틀부터 잡는것을 목표로 했습니다.
2. 각 개인의 GitHub Issue를 생성합니다.
2-1. Git Actions를 통해 secret파일들도 공유합니다.
4. 작업을 완료하면 각자의 생성한 브랜치에서 코드를 push하여 Pull Requset합니다.
5. 팀장이 코드를 확인하고 Merge합니다.
